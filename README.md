# DiscordBotWithPython


### 사용한 패키지


#### env 파일 사용을 위해

```shell

pip3 install python-dotenv 

```

#### HTTP 라이브러리

```shell

pip3 install requests

```

#### Discord 

```shell

pip3 install discord

```

### asyncio [비동기 프로그래밍을 위한 모듈이며 CPU 작업과 I/O를 병렬로 처리]

```shell

pip3 install asyncio
```
